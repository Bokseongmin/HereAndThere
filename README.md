# HereAndThere

**관광지 추천 웹 앱**

Team : MZ니어
Period : 2023.03.17 ~ 2023.03.29

---

## Description 

코로나 종식을 맞이한 만큼 관광수요가 늘며, 이에 따른 관광객의 목적에 따라 각 지역의 특색 명소나 관광지를 추천하는 웹 앱
